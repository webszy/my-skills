# Report Checklist Template

Use this template to ask the user for platform exports.

| Priority | Platform | Report | Configuration | Metrics | Purpose | Export format |
|---|---|---|---|---|---|---|
| P0 | Firebase / GA4 | Events | Date range, event name, key params | Event count, users | Identify available events | CSV |
| P0 | Firebase / GA4 | Core feature funnel | Code-confirmed steps | Users, conversion, elapsed time | Measure activation | Screenshot or CSV |
| P0 | AdMob | Date + ad unit | Date, ad unit, country if possible | Revenue, impressions, eCPM, CTR | Diagnose ad placement trends | CSV |
| P0 | App Store Connect | Source type + territory | Date, source type, territory | Views, downloads, conversion | Store acquisition | CSV |

## Priority Guidance

- P0: required for the main question.
- P1: needed for segmentation or confidence.
- P2: useful for deeper follow-up.
